# KCA Support Chatbot Frontend

## Description

A modern React and TypeScript frontend for the KCA Student Services Chatbot. The application provides students with a simple and interactive way to access information about admissions, finance, ICT support, academics, library services, and graduation processes.

## Features

* Floating chat assistant widget
* Real-time communication with a FastAPI backend
* Clean and professional user interface
* Responsive design
* Built with React, TypeScript, Tailwind CSS, and Axios

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Axios

This frontend is designed to integrate seamlessly with university websites and student portals, providing students with instant access to institutional information and support services.
